Variables

JavaScript es un lenguaje débilmente tipado, por lo que al declarar las variables no se indica su tipo. Además, el tipo de una variable puede cambiar varias veces a lo largo de la ejecución del programa.

Las variables pueden declararse con "var" o con "let". La diferencia está en que con "var", las variables tienen ámbito de función, pudiendo ser utilizadas desde cualquier punto de la función, o incluso antews de ser definidas.

Sin embargo, con "let" las variables tienen ámbito de bloque, por lo que solo pueden ser utilizadas a partir de su definición en el código.

Las constantes se definen con "const", se escriben siempre con mayúsculas y de les debe asignar un valor al declararlas. No se puede modificar el valor de una constante, por lo que si lo intentamos, se produce un error.

Funciones

Se declaran con la palabra reservada "function" y se les pasan los parámetros entre paréntesis. Pueden devolver un valor si se utiliza la palabra reservada "return" (si no tiene return, es como si devolviese undefined).

Hoisting

Las funciones pueden utilizarse antes de haberlas declarado, gracias a una característica de JavaScript llamada Hoisting, que permite utilizar variables y funciones antes de inicializarlas, e impide que esto nos dé error. En el caso de las variables, almacena el nombre en memoria y les asigna "undefined", y en el caso de las funciones, las almacena por completo.

Parámetros

Son los valores que se pasan a la función para que los utilice y realice una tarea concreta.

Si pasamos menos parámetros de los requeridos a una función, el valor de estos será "undefined".

Se pueden asignar valores por defecto a los parámetros, que obtendrán este valor si no se le pasa ninguno a la función.

En el caso de no saber cuántos parámetros va a recibir la función, podemos acceder a ellos mediante el array "arguments[]".

Las funciones en JavaScript son un tipo de dato más, por lo que se pueden pasar por argumento o asignarlas a una variable.

Funciones anónimas

Son funciones que se asignan a una variable, que pueden autoejecutarse o que se pueden asignar a un manejador de eventos.

Funciones anónimas

Son funciones anónimas declaradas de una forma más corta.

Características:
  • Se elimina la palabra "function".
  • Si la función solo tiene un parámetro, se pueden eliminar los paréntesis.
  • Se utiliza el símnbolo "=>".
  • Si el código del interior de la función solo ocupa una línea, se pueden eliminar las llaves y la palabra "return".
